pydra.engine.core module¶

Basic processing graph elements.

class pydra.engine.core.TaskBase(name: str, audit_flags: pydra.utils.messenger.AuditFlag = <AuditFlag.NONE: 0>, cache_dir=None, cache_locations=None, inputs: Union[str, pydra.engine.specs.File, Dict, None] = None, messenger_args=None, messengers=None, rerun=False)¶

Bases: object

A base structure for the nodes in the processing graph.

Tasks are a generic compute step from which both elemntary tasks and Workflow instances inherit.
